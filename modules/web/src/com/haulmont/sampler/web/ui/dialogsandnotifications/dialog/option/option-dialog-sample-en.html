An option dialog is a modal window that includes a message and one or more buttons which represent actions available to users.

The dialog can be invoked by showOptionDialog() method of the Frame interface. This interface is implemented by screen controllers, so these methods can be used directly in the controller code.

In addition to parameters described for showMessageDialog(), the method takes an array or a list of actions. A button is created for each action. After a button is clicked, the dialog closes and invokes the actionPerform() method of the corresponding action.

If you need an action with a standard name and icon, use an anonymous class derived from DialogAction. Five types of actions are supported: OK, CANCEL, YES, NO, CLOSE. Names of corresponding buttons are extracted from the m–∞in message pack.
