Компонент TabSheet имеет семь предопределённых стилей:

  • centered-tabs
  • compact-tabbar
  • equal-width-tabs
  • framed
  • icons-on-top
  • only-selected-closeable
  • padded-tabbar

Вы можете применить эти стили в XML-дескрипторе используя атрибут stylename или в контроллере экрана используя константы из класса HaloTheme.
