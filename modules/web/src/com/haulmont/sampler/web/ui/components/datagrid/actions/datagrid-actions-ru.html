Компонент DataGrid реализует интерфейс ActionsHolder и может содержать как кастомные действия, так и стандартные, определенные в перечислении ListActionType.

Если действие имеет заголовок, оно отображается в контестном меню DataGrid. Кроме того, действия DataGrid могут быть назначены кнопкам, расположенным в любом месте экрана.

С помощью XML элемента buttonsPanel в DataGrid можно добавить контейнер ButtonsPanel для отображения кнопок вверху DataGrid.
