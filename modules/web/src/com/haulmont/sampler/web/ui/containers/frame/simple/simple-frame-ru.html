Элемент frame предназначен для включения в экран фрейма.

Атрибуты:

  • src - путь к XML-дескриптору фрейма.

  • screen - идентификатор фрейма в screens.xml (если фрейм зарегистрирован).

В данном примере один и тот же фрейм (пример кнопки) включен в экран дважды: первый раз путем указания идентификатора фрейма из screens.xml, а второй раз с помощью указания пути к XML дескриптору.
