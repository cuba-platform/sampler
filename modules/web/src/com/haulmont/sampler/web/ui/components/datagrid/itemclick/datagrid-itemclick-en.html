The setItemClickAction() method allows you to define an action that will be performed when a DataGrid row is double-clicked. If such action is not defined, the DataGrid will attempt to find an appropriate one in the list of its actions in the following order:

  • The action assigned to the Enter key by the shortcut property

  • The edit action

  • The view action

The setEnterPressAction() allows to you define an action executed when Enter is pressed. If such action is not defined, a DataGrid will attempt to find an appropriate one in the list of its actions in the following order:

  • The action defined by the setItemClickAction() method

  • The action assigned to the Enter key by the shortcut property

  • The edit action

  • The view action
