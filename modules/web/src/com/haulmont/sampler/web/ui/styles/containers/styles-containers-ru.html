Компонент CssLayout имеет один предопределённый стиль - v-component-group.

Документация

Все остальные контейнеры имеют три предопределённых стиля, которые должны использоваться с атрибутом showAsPanel, который должен иметь значение true:

  • borderless
  • card
  • well
Документация

Вы можете применить эти стили в XML-дескрипторе используя атрибут stylename или в контроллере экрана используя константы из класса HaloTheme.
