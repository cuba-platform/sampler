addCustomField() method of the FieldGroup component is used together with the custom="true" attribute of field and it allows you to set your own field view. It takes two parameters: field identifier specified in the id attribute of field and the implementation of theFieldGroup.CustomFieldGenerator interface.

generateField() method of the CustomFieldGenerator interface is invoked by FieldGroup. A data source and field identifier, for which this generator is registered, are passed into the method. The method should return a visual component (or container), which will be displayed in the field.

In this example, we have created a LookupField with two options and used it to edit the boolean entity attribute.
