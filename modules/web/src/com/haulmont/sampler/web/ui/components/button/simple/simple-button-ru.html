Кнопка может содержать текст или пиктограмму (или и то и другое).

Атрибут icon задает путь к пиктограмме. Подробное описание правил расположения пиктограмм находится здесь.

С помощью атрибута description для кнопки можно задать всплывающую подсказку.

В атрибуте invoke задается имя метода контроллера, который вызывается при нажатии на кнопку. Если метод содержит параметр типа Component, то в него будет передан экземпляр вызвавшей его кнопки.
